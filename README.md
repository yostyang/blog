# 约斯特普通 · 个人博客

由 `约斯特普通-文章合集.epub` 生成的静态个人博客，共 179 篇文章，4 个专题。
视觉与排版参照「少数派 / 爱范儿」。

## 本地预览

```bash
cd blog
python3 -m http.server 8000
# 浏览器打开 http://127.0.0.1:8000
```

## 目录结构

```
blog/
├─ index.html          首页：顶部导航 + 焦点大图区 + 文章信息流（分类筛选/搜索）
├─ posts/*.html        每篇文章独立页面（居中阅读栏 + 上一篇/下一篇）
└─ assets/
   ├─ style.css        全站样式
   ├─ avatar.jpg       头像
   └─ images/          文章配图
```

## 设计

参照少数派 / 爱范儿：

- 顶部固定导航栏（头像 + 站名 + 分类导航 + 搜索）
- 焦点区：1 张大图卡片 + 2 张小图卡片
- 信息流：少数派式横向卡片（文字在左、缩略图在右、底部作者头像与日期）
- 文章页：居中阅读栏，作者信息条 + 干净图文排版
- 主色取少数派红 `#d23c2e`

头像取项目根目录的 `avatar.jpg`，生成时自动拷到 `assets/avatar.jpg`。

## 重新生成

内容改了 epub 后，在项目根目录重跑生成器即可（会自动带上 `avatar.jpg`）：

```bash
python3 build_blog.py 约斯特普通-文章合集.epub -o blog
```

## 部署

`blog/` 是纯静态站点，可直接托管到 GitHub Pages / Vercel / Netlify，
或飞书妙搭（把 `blog` 目录部署成公网链接）。
